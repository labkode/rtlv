What is a system ?

A system is a working application you want to monitor its logs. The application can be from a simple web application to the system log of the OS.

What format must have the logs

The log needs three fields to be accepted

system This is the id of system you are monitoring.
level This is theinformation level of the log. Valid values are info, warn and error
msg This is the message of the log
ts This is the timestamp of the log in Unix Epoch format(10 digit)

cURL samples

Create an info log curl -X POST realtimelogviewer.appspot.com/admin/logs --data '{"system":"labkodesys","level":"info","msg":"Hello world", "ts": 1430649856}'
